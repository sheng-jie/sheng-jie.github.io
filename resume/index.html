个人简历

个人简介

我是颜圣杰，.NET 技术爱好者，Microsoft MVP，热爱学习和写作，拥抱开源。一名立志成为架构师并为之努力奋斗的程序员。于2014年毕业于河南大学计算机专业，毕业以来一直从事软件开发工作，已有8年开发经验，擅长.NET Core、DDD、ABP、OAuth2.0、Microservice、Kubernets和DevOps。目前正在写个人的第一本技术书籍：《.NET 云原生》。

个人优势

  1. 8年.NET开发经验，4年ERP软件开发运维经验，3年以上项目管理经验。
  2. 熟悉企业内部业务，擅长系统整合，有企业数字化转型经验。
  3. 连续4年微软最有价值专家。
  4. 有企业中台设计和研发经验。
  5. 3年以上微服务开发实践，熟练掌握.NET 技术栈应用的开发。
  6. 热爱学习，善于总结与分享，持续有技术文章输出。

工作经验

深圳华强集团有限公司 | 2020.7-至今 | 部门主管

1. 项目名称：秒通平台

秒通平台是华强集团内部协同办公平台，分为Web端和移动端。内置OA、审批、考勤、绩效等常规业务模块，提供工作台集成其他内部系统，提供应用中心快速接入应用至移动端。

技术栈：.NET Core MySql Redis MongoDB RabbitMQ Hangfire Cap Apollo Skywalking ELK K8S TKE

工作内容

  1. 负责应用中心的设计开发
  2. 负责消息中心的设计开发
  3. 移动端混合式开发架构转型方案选定和推进
  4. 系统架构的完善
  5. 带领技术团队，推进公司数字化转型

工作成果

  1. 联合制定统一路由方案，并上线应用中心，完成Android/iOS 混合式架构转型，提高业务接入效率。
  2. 完成秒通平台的改造，实现企业内业务的整合和高效集成。

2. 项目名称：用户中台

用户中台，主要对诸如：统一认证、统一权限控制、统一组织员工管理等通用能力进行沉淀，解决企业内部重复功能建设问题，快速响应业务发展需要。

技术栈：.NET Core OAuth2.0 PgSQL Redis RabbitMQ Apollo DDD IdentityServer4 K8S TKE

工作内容

  1. 用户中台架构设计
  2. 多用户体系认证中心方案的设计及实现
  3. 统一权限控制模块的设计
  4. 组织员工模块的设计

工作成果

  1. 完成新旧认证中心的无缝切换，新系统按用户体系对接至认证中心，实现单点登录和认证隔离。
  2. 实现权限控制通用能力的沉淀，新项目可快速集成权限管理。

3. 项目名称：HR-薪酬-考勤-绩效 一体化

该项目包含两个新系统开发和两个旧有系统的整合，其中HR和薪酬系统为新系统。HR系统目标是替换掉旧有用友NC系统的人力资源管理模块，解决人事管理繁琐、数据不准的问题。薪酬系统主要是基于HR、考勤、绩效的基础上，构建智能薪酬，实现集团下多种发薪规则的按需配置，确保薪酬准时准确发放。

技术栈：.NET Core MySql MongoDb RabbitMQ Apollo Cap Hangfire Redis DDD K8S

工作内容

  1. HR系统人事档案的设计
  2. 入离调转业务的梳理及评审
  3. 数据归档方案的设计与评审
  4. 薪酬系统方案设计
  5. 薪酬动态公式方案的设计与实现

工作成果

  1. 薪酬系统成功上线
  2. HR系统完成开发并成功对接已有业务系统

  2. 小程序项目的开发和运维
  3. 硬件对接和联调
  4. 带领技术团体，推进项目任务

金蝶软件(中国)有限公司 | 2015.7-2019.5 | 研发工程师

技术栈：.NET SqlServer Redis

工作内容

  1. 负责供应链销售模块的开发和维护。
  2. 参与供应链的条码管理模块的设计、开发和维护。
  3. 独立设计预计可发量（供需平衡）模块。
  4. 性能和紧急问题的分析和解决。
  5. 负责第三方API对接，快递管家对接。
  6. 代码重构。

工作成果

  1. 多次荣获月度和年度优秀员工称号。
  2. 设计的《一种灵活把控即时进销存的解决方案》，并申请专利。
